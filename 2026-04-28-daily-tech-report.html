Bricks Builder XSS Vulnerability — Upgrade to 2.3 Now

Security

A reflected Cross-Site Scripting vulnerability (CVE-2026-41554, CVSS 7.1) affects Bricks Builder versions 1.9.2 through 2.2.x. The flaw allows unauthenticated attackers to execute malicious scripts via crafted URLs by exploiting unescaped user input reflected into HTML and JavaScript contexts. Upgrade to Bricks 2.3+ immediately.

Why it matters: If you or your clients run Bricks Builder, this is a critical action item. The vulnerability is unauthenticated and exploitable via crafted URLs — no login required. Update all Bricks sites to version 2.3 or later today. If patching is delayed, deploy a WAF with virtual patching and enforce strict CSP headers.
